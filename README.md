# Flutter X Widgets

一个精心打造的 Flutter 组件集合，致力于提供高质量、易用的 UI 组件。

## 环境要求

- Flutter 版本: 3.27.2
- 开发环境: iOS 优先（Android 版本正在优化中）
- 推荐设备: iPhone 6 模拟器

## 快速开始

1. 获取依赖

```bash
flutter pub get
```

2. 运行项目

- 选择 iPhone 6 模拟器
- 点击运行或使用命令行 `flutter run`

## 注意事项

- 当前版本主要针对 iOS 平台进行了优化
- Android 平台可能存在部分已知问题，我们正在积极修复中
- 建议使用 iPhone 6 模拟器以获得最佳体验
